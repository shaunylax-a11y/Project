# Project Summary

## Purpose
This project analyzes NBA player statistics across multiple seasons to understand scoring patterns, seasonal trends, and the relationship between usage percentage and scoring.

## Methods
The project uses Python, pandas, and matplotlib to inspect the data, summarize key variables, create visualizations, and join in additional team information using a lookup table.

## Main Findings
- Most player-season scoring averages are concentrated at the lower end of the distribution.
- Average points per game change over time across seasons.
- Usage percentage and points per game have a positive relationship.
- Joining team information adds useful conference and division context.

## Outputs
The repository includes:
- the original analysis notebook
- exported HTML and Markdown notebook versions
- a join script
- saved result tables
- saved visualizations
