# Project# NBA Player Statistics – Exploratory Data Analysis

## Dataset
This project uses the NBA Player dataset (`all_seasons.csv`).

## Source:  
(https://www.kaggle.com/datasets/salikhussaini49/nba-data)
The dataset contains player-level NBA season statistics, including points per game, rebounds, assists, usage percentage, team information, and season data across multiple years.

## Note: The raw dataset is not included in this repository and is intentionally excluded via `.gitignore`.

## Project Overview
The goal of this project is to explore and better understand trends in NBA player performance over time using basic exploratory data analysis (EDA).

This analysis includes:
- Inspecting the structure and size of the dataset
- Summary statistics for player performance
- League-wide scoring trends over time
- Visualizations of points per game and usage

## Files in This Repository
- `analysis.ipynb` – Jupyter Notebook containing the full analysis
- `analysis.html` – HTML export of the notebook
- `analysis.md` – Markdown export of the notebook
- `analysis_files/` – Images generated from the Markdown export
- `.gitignore` – Prevents raw data and system files from being uploaded
